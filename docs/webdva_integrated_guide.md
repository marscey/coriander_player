# 云服务集成文档

## 功能概述

Coriander Player 现已集成私有云服务支持，基于 WebDAV 协议实现云音乐播放和管理功能。

## 主要特性

### 1. 云服务连接管理
- 支持添加、编辑、删除 WebDAV 连接
- 连接状态测试和验证
- 连接信息本地持久化存储

### 2. 云文件浏览
- 类似文件管理器的界面
- 支持文件夹导航
- 音频文件识别和分类显示
- 文件大小、修改时间等信息展示

### 3. 音频播放
- 直接播放云端音频文件
- 自动下载到临时目录播放
- 播放完成后自动清理临时文件

### 4. 批量操作
- 下载文件到本地
- 添加到播放列表
- 扫描文件夹到音乐库
- 多选文件操作

## 使用指南

### 添加云服务连接

1. 点击左侧菜单栏的「连接」
2. 点击右上角的「+」按钮
3. 填写连接信息：
   - 连接名称：自定义显示名称
   - 服务器地址：WebDAV 服务器 URL
   - 用户名：登录用户名
   - 密码：登录密码
4. 点击「测试连接」验证配置
5. 点击「保存」完成添加

### 浏览云文件

1. 在连接列表中点击已添加的连接
2. 进入文件浏览器界面
3. 点击文件夹进入子目录
4. 点击音频文件直接播放

### 批量操作

#### 下载文件
1. 长按或右键点击文件/文件夹
2. 选择「下载」
3. 文件将下载到本地 Downloads 目录

#### 添加到播放列表
1. 选择多个音频文件（长按或点击选择框）
2. 点击右上角「添加到播放列表」
3. 文件将自动下载并添加到播放列表

#### 扫描到音乐库
1. 选择文件夹
2. 点击「扫描」
3. 文件夹中的音频文件将被添加到音乐库

## 技术实现

### 核心类

- `CloudConnection`: 云服务连接数据模型
- `CloudServiceManager`: 连接管理和持久化
- `WebDAVService`: WebDAV 协议实现
- `CloudAudioPlayer`: 云端音频播放
- `CloudScanner`: 云端文件扫描
- `CloudFileBrowser`: 文件浏览器界面

### 依赖库

- `http`: HTTP 请求处理
- `webdav_client`: WebDAV 客户端支持
- `shared_preferences`: 本地存储
- `xml`: XML 解析

## 配置示例

### Nextcloud WebDAV
- 服务器地址: `https://your-nextcloud.com/remote.php/dav/files/username/`
- 用户名: 你的 Nextcloud 用户名
- 密码: 你的 Nextcloud 密码或应用密码

### 坚果云 WebDAV
- 服务器地址: `https://dav.jianguoyun.com/dav/`
- 用户名: 你的坚果云账号
- 密码: 你的坚果云密码

### Synology WebDAV
- 服务器地址: `https://your-nas.com:5006/webdav/` (HTTPS)
- 用户名: 你的 DSM 用户名
- 密码: 你的 DSM 密码

## 注意事项

1. **网络要求**: 需要稳定的网络连接，大文件下载可能需要较长时间
2. **存储空间**: 播放时会临时占用本地存储空间
3. **隐私安全**: 连接信息本地加密存储，不会上传到任何服务器
4. **文件格式**: 支持 MP3, FLAC, WAV, AAC, M4A, OGG, OPUS, APE, WMA 等主流音频格式

## 故障排除

### 连接失败
- 检查服务器地址是否正确
- 确认用户名和密码
- 验证 WebDAV 服务是否启用
- 检查网络连接

### 文件无法播放
- 确认文件格式是否支持
- 检查文件是否完整
- 验证网络连接稳定性

### 下载失败
- 检查本地存储空间
- 确认网络连接
- 验证文件权限
